# Haggly Landing Page

Simple English landing page built with HTML/CSS and a minimal Node.js backend for signup.

## Included
- Orange background (`#F5621E`)
- Top menu with `About us` and `Help`
- Logo in the top-left
- Hero section with email signup
- Minimal backend that stores signups in CSV

## Run

```zsh
cd /Users/oscarwoldskaarderud/Desktop/haggly
npm start
```

Then open `http://localhost:8080`.

Signups are stored in `data/signups.csv`.

## Static-only option
You can open `index.html` directly in a browser, but signup saving will not work without the server.
